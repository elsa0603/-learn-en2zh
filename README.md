# Flutter English Reader

A simple bilingual (English + Chinese) reading app with English text-to-speech (TTS) playback.

## Features

- Article list with title and English preview
- Detail page shows **English** and **Chinese** text
- One-tap **Play English** button using `flutter_tts`
- Easy to extend by editing `assets/data/articles.json`

## Getting Started

1. Make sure you have Flutter installed.
2. Extract this project, then in the project root run:

   ```bash
   flutter pub get
   flutter run
   ```

3. To add more articles, edit `assets/data/articles.json` and rebuild the app.
